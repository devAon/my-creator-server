### 👩‍💻 개발환경

#### 기술스택 

- 프로그래밍언어: JavaScript 

- 백엔드개발프레임워크: Express 

- 데이터베이스: MySQL 


#### 개발툴 

- IDE : Visual Studio Code 

- DB 구축/운영: MySQL Workbench 

-  API 테스트: Postman 

#### 배포 

- AWS EC2 -클라우드환경컴퓨팅시스템 

- AWS RDS -클라우드환경데이터베이스관리 시스템 

- AWS S3 -클라우드환경객체스토리지
